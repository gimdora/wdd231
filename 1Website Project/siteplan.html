Seoul Walks Planner — Site Plan

W05 Individual Project Site Plan (living document)

Site Name

Seoul Walks Planner

I chose this name because it clearly communicates the purpose: planning walking routes in Seoul. The name is short, memorable, and works well for branding.

Optional domain idea: seoul-walks.example

Site Purpose

The site helps users discover and compare walking routes in Seoul. It provides route information (distance, time, difficulty, nearby subway, tags) and allows users to view details in a modal and save favorite routes using local storage.

The site will also display weather information to support planning like current conditions and a short forecast using a public API, and it will generate route cards dynamically from a JSON data source.

Scenarios

  • “I only have 45 minutes today. Which route near me is short and easy?”
  • “What walk is best for a night view, and how do I get there by subway?”
  • “It looks cold today. Which route is mostly sheltered or has indoor nearby spots?”
  • “How can I save my favorite routes and open them quickly later?”

Color Scheme

Primary: #0F2A43 (Deep Navy)

Used for header background, key buttons, and important headings.

Accent: #2B7BBB (Sky Blue)

Used for links, highlights, focus styles, and small UI accents.

Surface: #F4F6F8 (Light Gray)

Used as page background and card surfaces to keep content readable.

Typography

Font: Inter (Google Fonts)

  • Headings (h1–h3): Inter 800/600 for strong hierarchy and clarity.
  • Body text: Inter 400 for comfortable reading.
  • UI labels/buttons: Inter 600 for emphasis.
